# Welcome to the New Nebula Logger Documentation!

We're excited to announce the launch of our new documentation site, powered by Docusaurus!

## What's New?

### 🎨 Modern Design
Our documentation now features a clean, modern design with:
- Dark mode support
- Improved navigation
- Better mobile experience
- Faster load times

### 🔍 Powerful Search
We've integrated Algolia DocSearch for lightning-fast search across all documentation. Find what you need instantly!

<!--truncate-->

### 📚 Versioned Documentation
You can now browse documentation for different versions of Nebula Logger:
- v4.18.0 (Latest)
- v4.17.0
- And older versions

Use the version dropdown in the navbar to switch between versions.

### 📝 Enhanced Content
All documentation has been reorganized and enhanced:
- **Platform Guides** - Detailed guides for Apex, LWC, Aura, Flow, and OmniStudio
- **Feature Guides** - Deep dives into tagging, scenarios, and more
- **Best Practices** - Production-ready patterns and recommendations
- **Troubleshooting** - Common issues and solutions

### 🚀 Better Developer Experience
- Code examples with syntax highlighting
- Copy button for code blocks
- Interactive examples (coming soon)
- API reference with search

## Migration from Wiki

We're in the process of migrating content from our GitHub Wiki to this new site. If you notice any missing content, please:
- Check the [GitHub Wiki](https://github.com/jongpie/NebulaLogger/wiki) for the original content
- [Open an issue](https://github.com/jongpie/NebulaLogger/issues) to let us know what's missing
